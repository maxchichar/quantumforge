# QuantumForge

**Open-source PQC Migration & Crypto Scanner**

Helping developers and organizations safely transition from classical cryptography (RSA, ECC, SHA-1, etc.) to post-quantum algorithms (Kyber/ML-KEM, Dilithium/ML-DSA, etc.).

### Goal
Build a practical tool that scans codebases for quantum-vulnerable cryptography and assists with migration to Post-Quantum Cryptography (PQC).

### Current Status
- Early development stage (Learning phase with Go)
- Building step-by-step: CLI scanner → Report generator → Real PQC integration → Web dashboard

### Tech Stack
- Go (Golang)
- liboqs-go (for post-quantum algorithms)
- Open Quantum Safe

### Roadmap
- Project 1: Hello Quantum CLI
- Project 2: Single File Scanner
- Project 3: Directory Scanner
- Project 4: Report Generation
- Project 5: Real PQC Demo
- Project 6: Web Interface

Built while learning Go in https://www.learn2earn.ng
